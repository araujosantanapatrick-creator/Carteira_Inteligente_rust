# Carteira Inteligente de Investimentos em Rust

## Descrição

Projeto desenvolvido em Rust para simular uma carteira inteligente de investimentos, aplicando conceitos de programação segura, tratamento de erros e regras de negócio.

## Funcionalidades

* Cadastro de ativos financeiros
* Validação de dados utilizando `Result`
* Cálculo do valor investido
* Cálculo do valor atual da carteira
* Cálculo da rentabilidade percentual
* Tratamento de erros seguindo boas práticas da linguagem Rust

## Tecnologias Utilizadas

* Rust
* Cargo
* Git
* GitHub

## Estrutura do Projeto

```text
src/
 └── main.rs

Cargo.toml
Cargo.lock
README.md
```

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/araujosantanapatrick-creator/Carteira_Inteligente_rust.git
```

2. Entre na pasta do projeto:

```bash
cd Carteira_Inteligente_rust
```

3. Execute:

```bash
cargo run
```
